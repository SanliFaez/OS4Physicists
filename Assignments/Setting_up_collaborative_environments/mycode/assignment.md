# Open Science for Physicists, Lecture 2, Collaborative environments


### Tutorial format
**Assignment objective**

Create a working computer code for performing the doppler velocimetry measurement. The goal of this assignment is to analyze the recoded data from an orbiting source of mono-frequency sound to estimate:
1. The angular frequency of rotation
2. The average sound frequency and maximum doppler shift
3. Radius of the orbit in SI units

Students have to discuss in their teams on distribution of task, timing and how they are working together and collectively submit a merge request with the working code from their group and the necessary documentation for reproducing the experiment. 
You can use the incomplete code provided in this directory, or choose any other route to making a working code

After this session, representatives of each team clone the final repository on a fresh (not preconfigured) PC and should get the code running faster than the other teams.
The provided codes will also be compared for their seamless integration, modularity, and good documentation.

When submitting your merge request, your code must be put inside the [../projects](projects subfolder) with a unique title of your subdirectory.
